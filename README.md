# ao_client_pubnub
PubNub MQTT Client Subscriber

## Dependencies
- Requires Node.js to run
- Update the below keys prior to running 

```javascript
    pubnub = new PubNub({
        publishKey : 'demo',
        subscribeKey : 'demo'
    })
```

```sh
    npm install
```

## Execution

```javascript
    node subscribe.js
```
