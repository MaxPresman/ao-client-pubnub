# ao-client-pubnub
PubNub MQTT Client Subscriber

## Dependencies
- Requires Node.js to run
- Rename config-default.json to config.json and update the keys prior to running

```javascript
    pubnub = new PubNub({
        publishKey : 'demo',
        subscribeKey : 'demo'
    })
```


## Execution

```javascript
    node subscribe.js
```
